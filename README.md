## Projects

### [WiFaKey](https://github.com/conghau2308/Authentication_Service) — Biometric OAuth 2.1 & OIDC Server
An Authorization Server that replaces passwords with face biometric authentication, built on OAuth 2.1 and OpenID Connect standards. Features AES-256-GCM encrypted opaque tokens and token family-based reuse detection.

- [Backend](https://github.com/conghau2308/Authentication_Service) · [Frontend](https://github.com/conghau2308/Auth_Developer_Portal) · [Live](https://auth-developer-portal.vercel.app)
- Stack: Java Spring Boot, Spring Security, PostgreSQL, Redis, Next.js, TypeScript, Docker

### [CS Compiler](https://github.com/conghau2308/Compiler_Online) — Online Compiler for a C-like Language
A compiler for CS, a custom C-like language with type inference, implemented in Python with ANTLR4. Compiles source files through Lexing, Parsing, Semantic Analysis, and Java bytecode generation. Includes a FastAPI backend and a browser-based IDE built with Monaco Editor.

- [Backend](https://github.com/conghau2308/Compiler_Online) · [Frontend](https://github.com/conghau2308/frontend-compiler) · [Live Demo](https://frontend-compiler-beta.vercel.app)
- Stack: Python, ANTLR4, FastAPI, Next.js, Monaco Editor, Docker, Azure, GitHub Actions

### [WPM Mobile](https://apps.apple.com/vn/app/365-bcore/id6754451420) — Workforce Planning & Monitoring
A cross-platform mobile app for employee attendance tracking (QR code + GPS), task planning, and daily progress reporting. Developed during internship at TMTECH and shipped to production on the App Store.

- Stack: React Native, TypeScript, Expo, EAS Build
